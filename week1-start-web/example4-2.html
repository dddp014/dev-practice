float
  • 같은 선상에서 요소를 왼쪽/오른쪽 끝부터 정렬시키고자 할 때 사용한다.
  • 선택자를 공중에 띄워 새로운 레이어층을 만든다.
  • float을 적용한 자식 요소들의 width 합이 부모 요소의 width 보다 작아야 제대로 작동한다.
  • 만약 마지막에 div 요소가 온다면, float인 div들과 겹치게 된다.
left1 div
left2 div
right1 div
right2 div

clear
  • float에 대한 속성을 제어한다.
  • clear: both; 는 float: right; 와 float: left; 모두 제어하겠다는 의미이다.
  • float을 마지막으로 적용한 요소 다음에 clear를 적용한다.
  • 마지막에 div 요소가 오더라도, float인 div들과 겹치지 않는다.
left1 div
left2 div
right1 div
right2 div
div → clear: both;
