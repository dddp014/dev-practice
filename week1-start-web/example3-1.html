CSS Inline Style 인라인 스타일
- <body> 태그의 해당 요소 태그 안에 CSS 코드를 작성한다.
- 우선 순위가 가장 높지만, 코드 재활용이 좋지 않다.
- 테스트 용으로만 사용하고, 가급적 쓰지 않는 것이 좋다.
Inline Style div Inline Style span
CSS Internal Style 내부 스타일
- <head> 태그의 <style> 태그 안에 CSS 코드를 작성한다.
- Inline Style 다음으로 우선순위가 높다.
- 특정 페이지에서만 적용하기 위해서 사용한다.
- 많이 사용하지는 않는다.
Internal Style div Internal Style span
CSS External Style 외부 스타일
- <link> 태그로 CSS 파일을 불러온다.
- 우선순위는 가장 낮지만, 가장 많이 사용하는 방식이다.
- 협업에 우수하고, 여러 HTML에 동일한 스타일을 적용할 수 있다.
- 상대적으로 가독성이 높고 유지보수가 쉽다.

Cascading : CSS 적용 우선순위
  • 1. 순서 : 나중에 작성된 코드 > 먼저 작성된 코드
  • 2. 디테일 : e.g. header p > p
  • 3. 선택자 : style > id > class > type
  • 4. Inline Style > Internal Style > External Style
